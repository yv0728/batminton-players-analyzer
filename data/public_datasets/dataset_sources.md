# Public Badminton Dataset Sources

Use these sources to collect more clips and improve model accuracy.

- Kaggle: Badminton Dataset
  - https://www.kaggle.com/datasets/ardyhealthy/badminton-dataset
- Kaggle: Badminton Shot Dataset
  - https://www.kaggle.com/datasets/phucthaiv02/badminton-shot-dataset
- Kaggle: AI Badminton Dataset
  - https://www.kaggle.com/datasets/sayeedulhaq/ai-badminton-dataset

Recommended data balance for better accuracy:
- beginner: 100+ clips
- intermediate: 100+ clips
- pro: 100+ clips

Clip quality recommendations:
- one player clearly visible
- consistent camera angle
- minimum 720p where possible
- mixed rally speeds and movement patterns
